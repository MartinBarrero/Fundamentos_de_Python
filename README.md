## 📚 Modules

### 🐍 Python Fundamentals
Core Python concepts covered from scratch, refactored and documented in English following PEP 8 conventions.

| File | Topics |
|------|--------|
| `01_basics_and_data_types.py` | Data types, arithmetic operators |
| `02_variables_and_built_in_functions.py` | Variables, naming conventions, f-strings, built-in functions |
| `03_operators.py` | Arithmetic, string, comparison and logical operators |
| `04_strings.py` | String methods, slicing, formatting |
| `05_lists.py` | List methods, slicing, unpacking |
| `06_tuples.py` | Tuple methods, immutability, conversion |
| `07_sets.py` | Set methods, operations, conversion |
| `08_dicts.py` | Dictionary methods, fromkeys, type conversion |
| `09_conditionals.py` | if/elif/else, truthy/falsy, ternary operator, match/case |
| `10_loops.py` | while, for, break, continue, range, enumerate, zip, list comprehension |
| `11_functions.py` | Parameters, return values, args/kwargs, type hints, lambda |
| `12_classes.py` | OOP, inheritance, access modifiers, class methods, static methods |
| `13_exceptions.py` | try/except/else/finally, custom exceptions, raising exceptions |
| `14_modules.py` | Imports, aliases, standard library |
